# Chart

```react-chartjs-2```のグラフを動的に表示するコンポーネント．
Line, Barにのみ対応．

## Basic Usage

<Playground>
  <Chart
    IChartComponent={Line}
    labelsArgs={{ 
      startDate: new Date("2006-01-02"), 
      endDate: new Date(), 
      margin: 7,
    }}
  />
